블로그 하는 이유

January 16, 2018
testing blogging
Read this "Within 1 minute".

일단 여기서(기타 손쉬운 블로깅 매체들을 놔두고 ?!) 블로그를 하는 이유 부터 분명히 하자. 블로깅은 여타 SNS와는 다르게 짧은 메시지 위주 의견보다는 좀 더 긴 호흡의 글을 쓰기에 알맞다. 특히나 기술과 관련된 내용을 기술하기 위해서는 어느 정도 분량이 필요하다. 물론 국내에서 제공하는 블로깅 서비스도 있지만, 개발자 관점에서 자유롭게 수정할 기회를 주는 블로깅을 한 번 해보고 싶었다.

얼마나 이 블로그를 지속할지는 모르겠지만, 하는 동안에는 다음과 같은 주제에 대해 글을 써볼 예정이다.
